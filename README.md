# IF4073-Tugas4
## Cara Menjalankan
1. Jalankan `pip install -r requirements.txt` untuk menginstall seluruh dependencies yang dibutuhkan program.
2. Jalankan `python main.py` untuk menjalankan program.
3. Untuk klasifikasi dengan YOLOv8, perlu mendownload weight sehingga mungkin memerlukan waktu yang lama.
